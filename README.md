# Unit Testing Practice with Mocha and Chai

## How to use this

run ```npm install``` in your terminal to install dev dependencies

## Dev Dependences
* [Mocha](https://mochajs.org/)
* [Chai](https://www.chaijs.com/)
* [NYC](https://istanbul.js.org/)
